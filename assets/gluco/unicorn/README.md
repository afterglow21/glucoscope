# Unicorn Gluco assets

Special square illustrations used only for the local Unicorn Gluco encounter.

Trigger policy:
- Evaluate only the first successful latest-glucose fetch after a page load.
- The latest reading must be fresh and exactly 100 mg/dL.
- At most one Unicorn Gluco is selected per local calendar day.
- Historical readings are never scanned for an encounter.
- The mobile glucose-tab replacement illustration is intentionally deferred.
