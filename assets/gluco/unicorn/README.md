# Unicorn Gluco assets

Special illustrations used only for the local Unicorn Gluco encounter.

Trigger policy:
- Evaluate the first successful latest-glucose response and each new latest measurement received while the page stays open.
- The latest reading must be fresh and exactly 100 mg/dL.
- Never scan historical entries for an encounter.
- Do not repeatedly evaluate the same measurement.
- At most one Unicorn Gluco is selected per local calendar day.

Display assets:
- `unicorn-gluco-01` through `unicorn-gluco-10`: square Letter-tab and collection illustrations.
- `unicorn-gluco-peek-clover.png`: transparent 772×547 mobile glucose-tab illustration.
- The peek illustration is visible only while the current fresh latest reading is exactly 100 mg/dL, then immediately returns to the normal Gluco peek.
