About GlucoScope

Current Status現在の開発状況

GlucoScopeの現在地と、これから整えていくこと。

GlucoScopeは、GitHub PagesでFriends & Family Previewを公開しています。

Kazuma自身のNightscoutデータを使い、血糖値、グラフ、ふりかえり、グルコのお手紙を、
実際に動く形で見てもらえる段階まで進みました。

公開したまま完成とせず、言葉づかい、デザイン、安全方針、AIの境界、
プライバシー、データ連携を継続して整えています。

GlucoScopeは、身近な人からの声を受け取りながら、やさしく育てている途中のプロジェクトです。

GlucoScope is now available as a Friends & Family Preview on GitHub Pages.

Using Kazuma’s own Nightscout data, the preview provides working glucose, graph, reflection, and Gluco letter experiences.

Publishing is not the finish line. Wording, design, safety boundaries, privacy, AI behavior, and data integration continue to be reviewed carefully.

GlucoScope is still growing gently through feedback from the people who see and use it.
