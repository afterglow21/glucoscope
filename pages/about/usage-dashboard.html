Usage Dashboard使われ方の見える化

GlucoScopeが安心して続いていくために、AI利用と公開ページの状態を必要な範囲だけ確認する入口です。

現在のAIお手紙は、GitHub PagesからCloudflare Workerを呼び、Turnstileの安全確認後にOpenAIで生成します。

Workerには日次・時間帯別の生成上限と、月間費用を守るためのしきい値があります。血糖値の良し悪しを評価したり、個人を追跡したりするための仕組みではありません。

お手紙の保存は現在ブラウザ内キャッシュが中心です。別の人や別端末でも同じ保存済み分析を見られる共有KVキャッシュは、これから設計・実装します。

AI letters are generated by OpenAI through a Cloudflare Worker after a Turnstile safety check from the GitHub Pages site.

The Worker has daily and time-slot generation limits together with budget guardrails. It is not designed to judge glucose values or track individual people.

Letter caching is currently browser-local. A shared Cloudflare KV cache, allowing the same saved analysis to be reused across devices and viewers, is still planned.

Cache Status保存済みキャッシュ

現在はブラウザ内のlocalStorageが中心です。共有KVキャッシュは未実装で、別端末・別の人への共有はまだできません。

Cost Guard利用上限と費用

日次6回、同じ時間帯・同じモードは原則1回を目安に制御し、月間費用にも警告・停止しきい値を設定しています。
