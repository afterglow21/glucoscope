Usage & Sustainability

Usage Dashboard使われ方の見える化

GlucoScopeが安心して続いていくために、使われ方をやさしく確認する入口。

Usage Dashboardは、GlucoScopeの公開サンプルページがどのくらい使われているかを、開発者が落ち着いて確認するための入口です。

見る対象は、AIお手紙の呼び出し回数、保存済みキャッシュの利用、利用上限、Cloudflare Web Analyticsによるページ閲覧の全体傾向などです。

血糖値の良し悪しを評価したり、糖尿病とともに生きる人を追いかけたり、個人を責めるための分析には使いません。

The Usage Dashboard is an entrance for calmly checking how the GlucoScope public demo is being used.

It may include AI letter requests, saved-cache usage, usage limits, and high-level page trends from Cloudflare Web Analytics.

It is not for judging glucose values, tracking people living with diabetes, or creating pressure. It exists to keep GlucoScope safe, sustainable, and transparent.
