# 02. Philosophy

## Core Philosophy

GlucoScopeの中心にある考え方は、次の一文です。

> 血糖データは、あなたを責めるためではなく、今日を理解するための手がかりです。

GlucoScopeは、血糖値の良し悪しを単純に評価するアプリではありません。
糖尿病とともに生きる人が、自分の一日を落ち着いて振り返り、次の日を少し整えるための血糖みまもりパートナーです。

## Not a report card

血糖データは、成績表ではありません。

- 高かったから失敗
- 低かったから反省
- TIRが低かったからダメ

という見方をしません。

代わりに、GlucoScopeは次のように考えます。

- 今日はどんな流れだったのか
- どの時間帯に変化があったのか
- 明日に向けて、どんな手がかりがあるのか
- 自分を責めずに見返すなら、何が見えてくるのか

## Gentle reflection

GlucoScopeが提供したい体験は、厳しい分析ではなく、やさしい振り返りです。

データを見ることが怖くならないように。
うまくいかなかった日にも、開くのが嫌にならないように。
血糖値を見る時間が、少しだけ安心できる時間になるように。

## Design attitude

GlucoScopeのデザインや言葉づかいは、次の姿勢を大切にします。

- 責めない
- 怖がらせない
- 急かさない
- 断定しすぎない
- 数字だけではなく、生活の文脈を大切にする
- 医療判断の代わりではなく、理解を支える道具としてふるまう

## Product boundary

GlucoScopeは医療機器ではありません。
診断、治療判断、インスリン量の指示は行いません。

GlucoScopeは、日々の血糖マネジメントをやさしく振り返るためのサポートツールです。
