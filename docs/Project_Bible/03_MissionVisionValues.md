# 03. Mission / Vision / Values

## Mission

糖尿病とともに生きる人が、血糖データを責められることなく、やさしく振り返れる場所をつくる。

GlucoScopeは、日々の血糖データを、評価ではなく理解のための手がかりとして届けます。

## Vision

GlucoScopeは、血糖マネジメントをひとりで抱え込まないための、やさしい血糖みまもりパートナーを目指します。

将来的には、Nightscout、Libre、Dexcom、CSVなど複数のデータソースに対応し、AIパートナーglucoが日々の振り返りをやさしく支える体験を提供します。

ただし、AIは医療判断を置き換えません。
GlucoScopeは、医療者・本人・家族との会話を支えるための理解の道具です。

## Values

### 1. Kindness first

血糖データを見る人の気持ちを最優先にする。
不安や罪悪感を増やさず、安心して振り返れる表現を選ぶ。

### 2. No judgment

GlucoScopeは血糖値を見て、利用者を評価しない。
「よい/悪い」だけで片づけず、背景や流れを一緒に見る。

### 3. Clarity

複雑なデータを、できるだけ分かりやすくする。
ただし、分かりやすさのために医療的な意味を誇張しない。

### 4. Safety

診断、治療判断、インスリン量の指示はしない。
体調不良や緊急時は、主治医や医療機関への相談を促す。

### 5. Transparency

できること、できないこと、データ連携の限界、AIの限界を正直に伝える。

### 6. Sustainability

GlucoScopeが続いていくこと自体も価値と考える。
収益化を検討する場合は、倫理的で持続可能な形を優先する。

## Tagline

> **Understand today. Improve tomorrow.**

このタグラインは、GlucoScopeのすべての設計判断の中心に置く。
