# 🍀 GlucoScope Project Bible

> **Understand today. Improve tomorrow.**

このフォルダは、GlucoScopeの設計・開発・公開準備における正本です。

GlucoScopeは、糖尿病とともに生きる人が、血糖データを責められることなく、やさしく振り返るための血糖みまもりパートナーです。

## このBIBLEの使い方

GlucoScopeの作業を始める前、特に今日のタスクを決める前には、必ずこのProject Bibleを確認します。

確認順序は次の通りです。

1. Project Bible全体を確認する。
2. ロードマップを確認する。
3. 前回のタスクと完了状況を確認する。
4. 今日やるべき作業を、ロードマップに沿って整理する。
5. 優先順位を変える場合は、その理由を明確にする。

## 正本ルール

- このフォルダの内容を、GlucoScopeの最上位設計として扱う。
