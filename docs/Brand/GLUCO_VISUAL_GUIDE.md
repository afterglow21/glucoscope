## 1. Purpose / このガイドの目的

This document is the visual source of truth for **gluco（グルコ）**,
the official AI companion of GlucoScope.

グルコを新規生成、編集、トリミング、投稿素材化するときに、
別の犬や別キャラクターへ変わってしまうことを防ぎ、
どの場面でも「同じグルコ」に見える状態を守るためのガイドです。

優先順位は次の通りです。

1. キャラクターの同一性
2. ユーザーへ与える安心感
3. GlucoScopeの思想と安全性
4. シーンやレイアウトの見栄え

画面全体の華やかさよりも、
グルコがグルコであり続けることを優先します。

## 3. Character Identity Lock / 固定するキャラクター特徴

### Overall silhouette / 全体の形

- 小さく、やさしく、親しみやすい犬のマスコット
- 頭がやや大きく、体が小さいデフォルメ
- 手足は短めで丸く、やわらかい形
- 体は細長くせず、ふんわり丸みを持たせる
- ぬいぐるみではなく、犬らしさを残した絵本調のキャラクター
- 子どもにも怖さを与えない、安心できる存在感

### Fur color and markings / 毛色と白い模様

- 基本色は、やわらかいライトブラウンと白
- 額から顔中央、鼻の周り、口元へ幅広い白い模様が続く
- 首、胸、お腹、手足は白
- 顔の左右、耳、背中、体側はライトブラウン
- しっぽはライトブラウンを基本とし、先端は白
- 白と茶色の境界は硬い線ではなく、ふわふわした毛並みで自然につながる
- 黄色、濃いオレンジ、全身クリーム色へ変えない

### Face / 顔

- 大きく丸い黒い目
- 目には小さな白い光を入れる
- 目つきは鋭くせず、素直で安心できる印象
- 小さなこげ茶色の鼻
- 小さく、やさしく微笑む口
- 薄いピンクの頬
- 口や眉の演技を強くしすぎない
- 表情が変わっても、顔の基本形と白い模様は固定する

### Ears / 耳

- 大きめの立ち耳
- 三角形を基本とする
- 耳の内側は濃いブラウン
- 耳先と外側に、少し長く広がるふわふわした毛がある
- 垂れ耳、丸耳、小さすぎる耳へ変更しない
- 耳先の毛はグルコの重要な識別点として扱う

### Tail / しっぽ

- ふわふわで、やわらかくカーブする
- 先端は白
- 細い直線状、短すぎるしっぽ、巻きすぎたしっぽへ変更しない

## 4. Art Style Lock / 固定する画風

- 淡い水彩
- 絵本のようなやわらかさ
- パステル調
- 強い輪郭線を避ける
- 毛並みはふわっとした筆致で表現する
- 光はやさしく、影は薄くする
- 写真風、3Dレンダー風、リアルな動物画にしない
- 派手すぎるアニメ表現にしない
- 清潔感は保ちつつ、冷たい医療イメージに寄せない
- 背景は白、淡いミント、やさしいグリーンを基本とする
- GlucoScopeの濃紺UIやグリーンの光と組み合わせる場合も、グルコ自身の水彩感を失わない

---

## 5. Emotional Direction / 表情と空気感

グルコが届ける基本感情：

- 安心
- 見守り
- 小さな希望
- やさしい応援
- 一緒に振り返る気持ち
- そっとそばにいる温度

「やさしすぎるくらいでちょうどいい」を基準にします。

### Suitable expressions / 合う表情

- 小さく微笑む
- 少し首をかしげる
- 静かに見守る
- ほっとした顔
- 小さく手を振る
- 少し心配しながらも、怖がらせず寄り添う
- 一緒に考えている
- 小さな発見を喜ぶ

### Avoid / 避ける表情

- 鋭い視線
- 怒る、叱る
- 強く泣き叫ぶ
- 大げさな恐怖
- 勝ち誇る
- 数字の良し悪しを判定する
- 医師や医療従事者として指示する
- 過度にハイテンションな演技

## 6. Four-leaf Clover / 四葉のクローバー

四葉のクローバー🍀は、
安心・希望・小さな幸運の象徴です。

使い方：

- 手に持つ
- 背景に淡く浮かべる
- 小物や光のモチーフとして使う
- 緑の円形背景の中に配置する
- 新しい始まりや成長を表す場面で使う

単なる装飾として大量に散らさず、
場面の意味を支える象徴として使います。

## 7. What May Change / 新しくしてよいもの

同一キャラクターを維持したまま、
次の要素は新しく描き下ろして構いません。

- 表情
- ポーズ
- 視線
- 小物
- 衣装
- 背景
- 季節
- 時間帯
- シーン
- カメラ位置
- 構図
- 余白
- Instagram等の媒体に合わせた画面比率

顔、耳、目、毛色、白い模様、体形、しっぽは変更しません。

## 8. Generation Workflow / 新規生成の手順

1. 用途と画面比率を決める。
2. 最優先参照画像と、用途に近い既存画像を確認する。
3. 固定特徴を短く言語化する。
4. プロンプト内で「既存のグルコと同一キャラクター」と明記する。
5. 今回新しくする表情、ポーズ、背景だけを指定する。
6. 文字なしのキャラクターイラストを生成する。
7. 顔、耳、白い模様、毛色、体形、しっぽを確認する。
8. 正式な日本語、ロゴ、コピーは別工程で配置する。
9. 正式素材へ採用する前に、下記QAチェックを行う。

## 9. Canonical Generation Prompt / 標準生成プロンプト

次のテンプレートを基本として使います。

```text
GlucoScope公式AIパートナー「グルコ」を、
提供された既存のグルコ画像と同一キャラクターとして新規描き下ろしてください。

グルコは、糖尿病とともに生きる人のそばにいる、
やさしい小さな犬のAIパートナーです。
一般的なかわいい犬や別の犬種として再解釈せず、
参照画像の顔、毛色、耳、目、白い模様、体形、しっぽを維持してください。

【固定するキャラクター特徴】
- 頭がやや大きく、体が小さい、親しみやすい犬のマスコット
- ライトブラウンと白の、ふわふわした毛並み
- 額から顔中央、鼻の周り、口元へ続く幅広い白い模様
- 首、胸、お腹、手足は白
- 顔の左右、耳、背中、体側はライトブラウン
- 大きく丸い黒い目と、小さな白い光
- 小さなこげ茶色の鼻
- やさしく微笑む小さな口
- 薄いピンクの頬
- 大きめの立ち耳
- 耳の内側は濃いブラウン
- 耳先に外側へ広がる、少し長いふわふわした毛
- ふわふわしたしっぽ、先端は白

【固定する画風】
- 淡い水彩
- パステル調
- 絵本のようなやわらかさ
- 強い輪郭線を使わない
- 安心、希望、小さな幸運を感じる雰囲気
- 四葉のクローバーを意味のある象徴として使う
- リアルすぎず、派手すぎるアニメ調にもせず、既存画像の水彩感を維持する

【今回のシーン】
（ここへ、今回の表情・ポーズ・小物・背景を書く）

【構図と用途】
（縦長、正方形、全身、左寄せ、Instagram表紙、Aboutページ等を書く）

画像内に文字は入れないでください。
重要なのは画面全体の派手さではなく、
既存のグルコと同一キャラクターに見えることです。
```

## 10. Negative Prompt / 避ける内容

```text
垂れ耳、丸耳、小さすぎる耳、耳先のふわ毛がない、
全身が白、クリーム色、黄色、濃いオレンジ、
顔中央の白い模様がない、白と茶色の配置が違う、
テディベア、トイプードル、一般的な子犬、別犬種、
細長い体、長すぎる脚、リアルな犬、写真風、3Dレンダー、
派手すぎるアニメ、太く硬い輪郭線、鋭い目、攻撃的な表情、
怖い雰囲気、医師、看護師、医療判断をするキャラクター、
文字、ロゴの誤生成、崩れた日本語
```

Negative Promptに対応していない生成環境では、
標準プロンプトの末尾へ「避けること」として追記します。

## 11. QA Checklist / 正式採用前の確認

### Character identity

- [ ] 既存のグルコと同じ顔に見える
- [ ] 額から口元まで白い模様が続いている
- [ ] 白とライトブラウンの配置が合っている
- [ ] 目が大きく丸く、白い光が入っている
- [ ] 鼻が小さく、こげ茶色
- [ ] 頬が薄いピンク
- [ ] 耳が大きめの立ち耳
- [ ] 耳の内側が濃いブラウン
- [ ] 耳先に長めのふわ毛がある
- [ ] 頭がやや大きく、体が小さい
- [ ] しっぽがふわふわで、先端が白い

### Style and feeling

- [ ] 淡い水彩・絵本調になっている
- [ ] 強い輪郭線や写真風表現になっていない
- [ ] 安心感があり、怖さや圧がない
- [ ] グルコがユーザーを評価・叱責していない
- [ ] 四葉のクローバーが意味を持って使われている
- [ ] GlucoScopeの画面や背景と調和している

### Production

- [ ] 不要な個人情報や健康情報が入っていない
- [ ] Nightscout URLやAPI情報が入っていない
- [ ] 日本語の誤字・崩れがない
- [ ] 文字を別工程にする必要がないか確認した
- [ ] 用途に合う余白と画面比率になっている

一つでもキャラクター同一性に重大な違和感がある場合は、
見栄えが良くても正式素材として採用しません。

## 12. Expression Icons / 表情アイコン

- 既存画像がある場合は既存の表情を優先する
- 不足する表情だけ新規生成する
- 正方形で統一する
- 淡いグリーンの円形クローバー背景を基本とする
- 顔の中心ではなく、円形背景を基準にトリミングする
- 画像内に文字を入れない
- 表情を強くしすぎず、既存のグルコらしさを守る

---

## 13. Social and Key Visuals / SNS・キービジュアル

Instagram等の重要素材では、
次の二段階制作を基本とします。

1. グルコと背景を、文字なしで高品質に新規生成する
2. 正式な日本語、英語コピー、ロゴ、UIスクリーンショットを別工程で配置する

これにより、
AI生成文字の誤字や崩れを避けながら、
キャラクターとレイアウト双方の品質を高めます。

GlucoScopeの濃紺UI、緑の光、白い余白を組み合わせても構いません。
ただし、グルコの色や水彩表現を濃紺UIへ無理に合わせて変更しません。

## 14. Final Principle / 最後の原則

グルコは単なる装飾ではありません。

GlucoScopeの思想、
カズマが大切にしてきた愛犬の記憶、
糖尿病とともに生きる人へ届けたい安心感を背負う、
大切な公式キャラクターです。

新しい作品を作るたびに、
「かわいいか」だけでなく、

**この子は、いつものグルコに見えるか。**  
**この表情は、見る人を責めずにそばにいられるか。**

を確認します。
